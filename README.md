**Overview**
This project analyses vehicle CO₂ emissions data using statistical and machine learning regression techniques. The primary method is Ordinary Least Squares (OLS) Linear Regression, with additional comparisons against Decision Tree, Random Forest, and Support Vector Regression models.

The project follows a structured data science workflow, Exploratory Data Analysis, Statistical Modelling, Model Validation, and Comparative Evaluation, making it well-suited as a foundation for understanding how quantitative methods can be applied to environmental data.
